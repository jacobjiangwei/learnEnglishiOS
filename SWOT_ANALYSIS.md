## 二、目标用户画像

### 2.1 核心用户群

#### 👦 小学生家长（K1-K6）
- **痛点**：孩子英语刚起步，不知道怎么辅导，各种辅导班费用高
- **需求**：课本同步练习、趣味互动、家长可追踪进度
- **付费意愿**：家长代付，价格敏感但为教育舍得投入
- **规模**：中国在校小学生约 1.08 亿，英语学习渗透率高

#### 🧑‍🎓 初高中学生（G7-G12）
- **痛点**：应试压力大，题目做完没有好的错题分析，单词总是记了忘
- **需求**：课本配套题库、考试突击（中考/高考英语）、智能错题重练
- **付费意愿**：家长代付，核心看提分效果
- **规模**：中国在校中学生约 8800 万

#### 🎓 大学生 / 考研 / 出国党
- **痛点**：四六级/考研/托福/GRE 需要大量刷题，市面工具分散
- **需求**：一站式练习 + 词汇积累 + 口语训练 + 写作批改
- **付费意愿**：自费，对效果和效率要求高，愿意为"真正有用"付费
- **规模**：每年四六级考考生约 2000 万人次，考研 400+ 万

#### 🌏 海外华人 / 华裔
- **痛点**：华人家长希望孩子学好英语，但找不到中文解释的专业英语学习工具
- **需求**：中英双语解释、适合 ESL 学习者的分级内容
- **付费意愿**：美元定价接受度高
- **规模**：海外华人约 6000 万，有子女教育需求的约 30%

### 2.2 次要用户群

| 用户类型 | 场景 | 规模 |
|---------|------|------|
| 职场白领 | 外企面试备考、日常邮件/汇报英语 | 数千万 |
| 英语教师 | 查词备课、给学生推荐题库 | 数百万 |
| 兴趣学习者 | 出国旅游、看美剧想提升 | 数千万 |

## 三、用户为什么会选择 Volingo

### 3.1 核心价值主张

> **一个 App 覆盖从小学到 GRE 的全程英语学习，用 AI 构建"查词→记忆→练习→测评"学习闭环**

### 3.2 用户选择 Volingo 的 5 个理由

**1. 教材同步、精准匹配**
- 绑定人教版等授权教材，按课本单元出题
- 不是泛泛的"英语练习"，而是"你这学期第三单元的针对性训练"
- **市面上几乎没有同类产品做到按教材单元 AI 批量生成高质量题库**

**2. 真正的全题型覆盖**
- 12+ 种交互题型（选择、填空、阅读理解、听力、口语、翻译、改写、纠错、排序、情景对话……）
- 竞品通常只做 1-3 种题型（多邻国偏选择题、百词斩偏单词、流利说偏口语）
- 一个 App 替代多个 App

**3. FSRS 科学记忆算法**
- 不是简单的"艾宾浩斯"固定间隔，而是 FSRS（Free Spaced Repetition Scheduler）
- 根据每次复习表现动态计算最优复习时间
- 同时应用于生词本复习 + 错题重练，双维度记忆强化

**4. AI 驱动的个性化**
- AI 兜底词典：任何生僻词都能查到（OpenAI 实时生成词条）
- AI 批量出题：基于教材内容自动生成题目，题量充足（每册 500-1300 题）
- AI 质量把控：自动校验答案合理性 + 人工抽检

**5. 全学段覆盖的持续陪伴**
- 小学 → 初中 → 高中 → 四六级 → 考研 → 托福/GRE
- 用户不需要换 App，一个账号跟随整个学习生涯
- **这是极强的用户粘性设计**——六年级用得好的小学生，升初中还会继续用

### 4.2 Volingo 的差异化壁垒

1. **教材级题库** — 竞品做不到按人教版单元精准出题
2. **全题型一站式** — 替代 3-4 个 App 的组合（查词 App + 背单词 App + 口语 App + 刷题 App）
3. **AI 内容管线** — 教材 PDF → Document Intelligence → GPT-4o 分析 → 批量出题，成本低（$3/册），可快速扩展
4. **双系统 FSRS** — 生词本和错题本都用 FSRS 调度，不是简单的"再做一遍"

## 五、SWOT 分析

| # | 优势 | 详细说明 |

| S3 | **教材精准对齐** | 按人教版等教材的单元、词汇、语法点出题，"课本的最佳伴侣"定位清晰 |

| S7 | **技术栈现代化** | SwiftUI + .NET 8 + Azure Cosmos DB + GPT-4o，架构清晰可扩展 |
| S8 | **全学段覆盖** | 小学到 GRE 一条龙，用户生命周期可长达 10+ 年 |

### 🔴 Weaknesses（劣势）

| # | 劣势 | 影响 | 缓解策略 |
|---|------|------|---------|
| W1 | **写作模块未完成** | 无法满足高年级用户的写作训练需求，缺少一个核心能力 | 优先开发，接入 GPT-4o 批改 |
| W2 | **无用户认证系统** | 无法做云端数据同步、换设备丢失数据、无法做用户裂变 | 接入 Firebase Auth（Apple/Google 登录） |
| W3 | **无付费/订阅系统** | 目前完全免费，没有收入来源，无法验证商业模式 | IAP 订阅接入是上架前必须完成的 |
| W4 | **品牌和用户认知为零** | 新产品冷启动，没有品牌积累 | 产品上架后需要 ASO + 内容营销 |
| W5 | **团队规模限制** | 单人/小团队，开发速度受限，需兼顾前后端 + 运营 | 优先核心功能，MVP 先行 |
| W6 | **仅 iOS 平台** | 丢失大量 Android（特别是中国市场 70%+ Android 用户） | Phase B 考虑 Flutter/Android |
| W7 | **场景对话模块未完成** | 实用口语训练的核心场景缺失 | 后续迭代中优先级较高 |
| W8 | **AI 内容质量波动** | GPT 生成的题目可能有少量质量问题 | 自动校验 + 10% 人工抽检 + 用户举报机制（已有） |
| W9 | **无社交/社区功能** | 缺少用户粘性的社交属性 | 后期加入学习小组、排行榜 |

```
                  ┌─────────────────────────────────────┐
                  │         Volingo 竞争力金字塔          │
                  ├─────────────────────────────────────┤
                  │                                     │
                  │          🏆 护城河层                  │

                  │          🎯 差异化层                  │
                  │    12+ 题型全覆盖 × 教材单元精准匹配     │
```

| P2 | **学习提醒推送** | 用 Push Notification 拉回用户，提升留存 |
| P2 | **题库扩充**（覆盖更多教材版本） | 北师大版、外研社版等，扩大用户基数 |

### Phase 2：增长与差异化（3-6 个月）
